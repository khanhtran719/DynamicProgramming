# DynamicProgramming
# Input 
N M  
Nhap khoi luong mon hang thu i: W[i] (i < n)  
Nhap gia tri mon hang thu i: V[i] (i < n)  
# Output
Cac mon hang duoc chon: ...  
Tong gia tri cua cac mon hang cho vao tui: ...
